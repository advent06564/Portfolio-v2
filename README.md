# Portfolio-v2

[![CI](https://github.com/advent06564/Portfolio-v2/actions/workflows/ci.yml/badge.svg)](https://github.com/advent06564/Portfolio-v2/actions/workflows/ci.yml)

Personal portfolio website built with HTML, CSS, and JavaScript. Features responsive design, project showcase, skills section, and contact form.

## Features

- **Responsive Design** — Adapts to any screen size
- **Project Showcase** — Display latest projects with descriptions
- **Skills Section** — Technical skills and expertise
- **Contact Form** — Direct messaging from the site
- **Interactive Elements** — Dynamic content via JavaScript

## Tech Stack

- HTML5
- CSS3 (animations, transitions)
- JavaScript (vanilla)
- Google Fonts
- FontAwesome icons

## Getting Started

### Prerequisites

- A modern web browser

### Run Locally

```bash
git clone https://github.com/advent06564/Portfolio-v2.git
cd Portfolio-v2
open index.html
```
